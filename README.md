# Fraud Detection System
A console-based fraud detection system built in Java using core data structures to detect suspicious financial transactions.

## About
This project demonstrates how data structures can be used to detect fraud in financial systems.  
The system analyzes transactions in real time and flags suspicious activities based on predefined fraud detection rules.

## Data Structures Used
- **Linked List** – Stores transaction history
- **HashMap** – Fast user lookup
- **Max Heap** – Displays highest risk fraud alerts
- **Graph** – Detects fraud rings between users
- **Queue (Sliding Window)** – Tracks recent transactions

## Features
- Add and delete users
- Process transactions with real-time fraud detection
- Automatically flag suspicious transactions
- Detect fraud rings using graph traversal
- Monitor the last 5 transactions per user

## How to Run
1. Open the project in **IntelliJ IDEA** or any Java IDE
2. Run `Main.java`
3. Sample users and transactions will load automatically
